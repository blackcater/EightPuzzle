## 运行

```shell
> git clone https://github.com/blackcater/EightPuzzle.git # 拷贝项目

> cd ./EightPuzzle-master

> npm run serve # 启动react客户端

> cd ./server # 在另一个命令行中

> npm start # 启动express服务器
```
